# EDA Project

This repository contains exploratory data analysis (EDA) work using Python libraries such as **NumPy**, **Pandas**, **Matplotlib**, and **Seaborn**.  
The focus is on learning and applying fundamental data manipulation and visualization techniques.

---

##  Contents
- **Numpy/**  
  Examples and practice files demonstrating NumPy operations.

- **Pandas/**  
  Work on Pandas including the Big Basket dataset and exercises on filtering, sorting, renaming, and handling missing values.

- **Matplotlib.ipynb**  
  Notebook with examples of data visualization using Matplotlib.

- **seaborn.ipynb**  
  Notebook with examples of advanced statistical visualizations using Seaborn.

- **.DS_Store**  
  System file generated by macOS.

- **README.md**  
  Documentation for the repository.

---

##  Key Topics Covered

### NumPy Basics
- Array creation and manipulation  
- Mathematical operations  
- Indexing and slicing  

### Pandas Basics
- Selecting rows and columns  
- Filtering with conditions  
- Sorting with `.sort_values()`  
- Renaming columns with `.rename()`  
- Checking and handling missing values with `.isnull()`, `.dropna()`, and `.fillna()`  

### Big Basket Dataset Analysis
- Practical application of Pandas methods  
- Data cleaning and preparation  
- Exploratory insights  

### Matplotlib Basics
- Core plotting library for Python  
- Line plots, bar charts, scatter plots, histograms  
- Customization of axes, labels, titles, and colors  
- Best for **low-level, customizable visualizations**

### Seaborn Basics
- Built on top of Matplotlib for **statistical data visualization**  
- High-level functions for box plots, violin plots, pair plots, heatmaps, etc.  
- Automatic handling of themes, palettes, and categorical comparisons  
- Best for **quick, clean, and insightful visualizations**
